Variables

brutalityTier:Int

Brutality tier for mobs (only used for some bosses)

chromaColor:Int

Color the Razer peripherals turn to when entering the level.

doorColor:Int

Color of the door leading to the level

loadingColor:Int

Not used atm.

loadingDescColor:Int

Not used atm.

musicIntro:String

File path to the music's intro

musicLoop:String

File path to the music's main loop

survivalTier:Int

Survival tier for mobs (only used for some bosses)

timedBlueprint:String

Blueprint name, as specified in column id in table item or column item in table weapon in CDB

timedDoor:Float

Time in minutes for the time door to definitely close

timedGoldMul:Float

Gold multiplier for gold behind timed door

timedScrolls:Int

Scrolls number behind timed door

wind:Float

Wind force used to move the hero scarf

zDoorColor:Int

Color of the Z door leading to the level
